# hayhooks

Arch package for the Hayhooks service layer on top of Haystack.

## Defaults

- bind: `127.0.0.1:1416`
- pipelines dir: `/var/lib/hayhooks/pipelines`
- env file: `/etc/hayhooks/hayhooks.env`

## Local install

Build the dependency packages first, then install `hayhooks` and enable the service.
